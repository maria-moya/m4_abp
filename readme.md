## Proyecto ABP Módulo 4

Proyecto consiste en desarrollar una aplicación en consola que permita realizar operaciones aritmeticas básicas solo utilizando JavaScript como:
- Estructura condicionales.
- Variables.
- Arreglos y ciclos.
- Funciones.

## Funcionalidad del código

El programa que se realizo solicita dos números al usuario a traves de un promt, valida que ambos sean valores numéricos y luego realiza cuatro operaciones aritméticas basica que son la suma, resta, multiplicación y división. 
Los resultados de esta operaciones se almacenan en un objeto, se muestran en la consola y al final filtra los resultados que son positivos.
